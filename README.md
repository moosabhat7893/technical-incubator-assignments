# Technical Incubator Assignments

This repository contains solutions to advanced Data Structures and Algorithms problems designed to meet industry-level performance and optimization standards.

These problems simulate real-world engineering scenarios such as autocomplete systems, network vulnerability detection, and performance monitoring systems.

---

## Problems Included

| Folder | Problem | Data Structure / Algorithm | Complexity |
|------|---------|----------------------------|------------|
| 01-streaming-max | Streaming Max Analytics | Deque (Monotonic Queue) | O(N) |
| 02-autocomplete-system | Global Autocomplete System | Trie | O(L) |
| 03-find-bridges | Network Vulnerability Detection | Graph + Tarjan's Algorithm | O(V+E) |
| 04-segment-tree | Range Performance Monitor | Segment Tree | O(log N) |
| 05-bitmask-dp | Optimal Resource Allocation | Bitmask Dynamic Programming | O(2^N × N²) |
